# 🚗 KaayDem

## Description

KaayDem est une plateforme de covoiturage développée en PHP dans le cadre du module de Programmation Orientée Objet (POO) à l'Institut Supérieur de Management (ISM).

L'application permet aux utilisateurs de publier des trajets, de réserver des places, de gérer les réservations, les véhicules et les évaluations des conducteurs. Le projet est développé selon l'architecture MVC et utilise une base de données MySQL avec PDO.

---

## Fonctionnalités

- Inscription et connexion des utilisateurs
- Gestion des administrateurs, conducteurs et passagers
- Création et gestion des trajets
- Réservation et annulation de réservations
- Gestion des véhicules
- Évaluation des conducteurs
- Gestion des statuts des trajets et des réservations

## Installation

1. Cloner le dépôt GitHub :

```bash
git clone https://github.com/Yafatou987/KaayDem.git
```

2. Ouvrir le projet avec Visual Studio Code.

3. Importer le fichier `kaaydem.sql` dans phpMyAdmin.

4. Vérifier les paramètres de connexion dans le dossier `config`.

5. Démarrer Apache et MySQL avec XAMPP.

6. Ouvrir le projet dans votre navigateur.
